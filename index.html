Introduction

Python is an open source development project that allows a community of users and contributors to make their software available under open source license terms.

This allows Python users to collaborate with eachother and exchange ideas and benefit from eachothers solutions to problems. This collaboration is key to the further development of Python as a development language.

This is a very brief intro into what Python is and how to interact with some of its features.

What You Should Know?

Variables

Python is able to use variables in a similar fashion to other programming languages. Here is an example:

a = [1, 2, 3]

b = [1, 2, 3]

c = a

print(a == b) #True

print(a is b) #False

print(a is c) #True

Modules

A module allows you to logically organize your Python code. Grouping related code into a module makes the code easier to understand and use. A module is a Python object with arbitrarily named attributes that you can bind and reference.

Simply, a module is a file consisting of Python code. A module can define functions, classes and variables. A module can also include runnable code.
